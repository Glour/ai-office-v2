# Presentation Creator - Создание презентаций через Markdown

## 📦 Установка

1. Скопируйте папку `skills/presentation/` в ваш workspace OpenClaw
2. Установите Marp CLI: `npm install -g @marp-team/marp-cli`
3. OpenClaw автоматически подхватит скилл при следующем сообщении
4. Проверка: спросите агента "Сделай презентацию про X"

## 🎯 Что умеет

- Создаёт профессиональные презентации из текста или outline
- Экспорт в PDF (универсальный), PPTX (PowerPoint), HTML (интерактивный)
- Использует Marp - Markdown для слайдов
- Поддержка изображений, кода, таблиц
- Встроенные темы и кастомизация

## ⚙️ Как работает

Скилл превращает ваш текст в Markdown-файл с синтаксисом Marp, затем экспортирует в нужный формат:

### Синтаксис Marp

**Структура слайда:**
```markdown
---
marp: true
theme: default
paginate: true
---

# Заголовок презентации
Подзаголовок или автор

---

## Слайд 2
- Пункт 1
- Пункт 2

---

## Слайд с изображением
![bg right:40%](image.jpg)
Текст слева от картинки
```

**Фоновые изображения:**
- `![bg](image.jpg)` - на весь слайд
- `![bg left](image.jpg)` - левая половина
- `![bg right:40%](image.jpg)` - правая 40%
- `![bg contain](image.jpg)` - вписать по размеру
- `![bg cover](image.jpg)` - заполнить с обрезкой

**Темы:**
- `default` - классическая
- `gaia` - современная
- `uncover` - минималистичная

**Центрированный титульный слайд:**
```markdown
<!-- _class: lead -->
# Заголовок
```

### Экспорт

```bash
# PDF (лучшее качество, универсальный)
marp presentation.md -o output.pdf

# PowerPoint (можно редактировать)
marp presentation.md -o output.pptx

# HTML (интерактивные слайды в браузере)
marp presentation.md -o output.html
```

## 💡 Примеры использования

**Пример 2:** "Создай питч-дек для стартапа: проблема, решение, модель монетизации, команда"

## 🔧 Настройка под себя

### Стиль и темы
- Измените `theme:` в frontmatter (`default`, `gaia`, `uncover`)
- Добавьте кастомные цвета: `<!-- _backgroundColor: #123 -->`, `<!-- _color: white -->`
- Используйте `class: lead` для центрированных слайдов

### Формат вывода
- PDF - для печати и универсального просмотра
- PPTX - если нужно дальше редактировать в PowerPoint
- HTML - для интерактивных онлайн-слайдов

### Советы по дизайну
- 1 идея = 1 слайд
- Максимум 6 пунктов на слайд
- Используйте изображения для визуального эффекта
- Крупные шрифты (дефолтные в Marp - хорошие)
- Не жёлтый на тёмном фоне, лучше белый или светло-голубой

### Расширенные возможности
- Добавьте заметки докладчика: используйте `---` и секцию notes
- Таблицы: стандартный Markdown-синтаксис
- Код: блоки с подсветкой синтаксиса
- Математика: поддержка KaTeX/MathJax
