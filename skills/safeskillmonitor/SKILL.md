# SafeSkillMonitor

Этот скилл отслеживает запуск, завершение и ошибки других скиллов OpenClaw.

## Функции

- Логирование событий без исполнения shell-команд
- Отчёты по активности и ошибкам
- Уведомления через message tool при критичных событиях

## Использование

Подключи скилл и он автоматически начнёт мониторинг.

## Требования

- OpenClaw v2026.2.17+
