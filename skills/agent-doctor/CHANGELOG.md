## [1.0.0] - 2026-03-06

### Added
- 🏥 Начальный релиз скилла Agent Doctor
- 7 блоков диагностики:
  - 🧠 Память (SQLite, WAL, embeddings, файлы)
  - ⏰ Кроны (статус, ошибки, расписание)
  - ⚙️ Конфиг (валидность, модели, плагины)
  - 📁 Файлы (SOUL, USER, skills)
  - 🔧 Gateway (статус, логи, порты)
  - 💾 Система (Node, Python, диск)
  - 🛡️ Безопасность (bind, auth, ключи)
- База знаний 28 типовых проблем с решениями
- Bash-скрипт автодиагностики (auto-diagnostic.sh)
- Примеры использования для 10 сценариев
- Две версии SKILL.md:
  - Рабочая (с локальной спецификой)
  - Публичная (универсальная для публикации)

### Features
- Безопасный режим: только чтение, фиксы после подтверждения
- Детальные отчеты с приоритетами проблем
- Оценка рисков для каждого фикса
- Выборочное применение фиксов (по номерам)
- Поддержка всех платформ (macOS, Linux, Windows)

### Documentation
- README.md - краткое описание
- SKILL.md - полный скилл для агента
- SKILL-public.md - универсальная версия
- EXAMPLES.md - 10 сценариев использования
- PROBLEMS_DATABASE.md - 28 проблем и решений
- CHANGELOG.md - история изменений

### Триггеры
- "продиагностируй себя"
- "самодиагностика"
- "проверь систему"
- "что с тобой не так"
- "диагностика агента"
- "self-check"
- "health check"

### Known Issues
- Нет автоматической проверки версий зависимостей
- Нет интеграции с мониторингом (Grafana, Prometheus)
- Bash-скрипт не проверяет embeddings (требует агента)
