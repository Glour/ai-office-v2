# 🏥 Agent Doctor

Комплексная самодиагностика OpenClaw агента. Проверяет все критичные системы и предлагает конкретные решения проблем.

## Файлы

- **SKILL.md** - внутренняя версия (с локальной спецификой)
- **SKILL-public.md** - универсальная версия для публикации (без личных данных)

## Что проверяет

1. 🧠 **Память** - SQLite, WAL mode, embeddings, файлы
2. ⏰ **Кроны** - статус, ошибки, последние запуски
3. ⚙️ **Конфиг** - валидность, модели, плагины
4. 📁 **Файлы** - SOUL.md, IDENTITY.md, skills
5. 🔧 **Gateway** - статус, логи, порты
6. 💾 **Система** - Node.js, Python, диск
7. 🛡️ **Безопасность** - bind, auth, ключи

## Использование

```
"продиагностируй себя"
"самодиагностика"
"health check"
```

## Безопасность

- ТОЛЬКО чтение и диагностика
- Фиксы применяются ТОЛЬКО после явного подтверждения
- Каждый фикс с объяснением и оценкой риска

## Типовые проблемы

1. WAL mode отключен → агент не видит новые записи
2. memory-core отключен после обновления
3. Нет embedding ключа
4. Gateway bind = 0.0.0.0 (опасно!)
5. Старая версия Node.js

## Когда запускать

- После обновления OpenClaw (обязательно!)
- При странном поведении
- Раз в неделю для профилактики
- Перед стрим-сессией
