# Tweet Writer - Вирусные твиты и треды для X 🐦

## 📦 Установка

1. Скопируйте папку `skills/tweet-writer/` в ваш workspace OpenClaw
2. OpenClaw автоматически подхватит скилл при следующем сообщении
3. Проверка: скажите агенту "Напиши твит про X" или "Создай тред про Y"

## 🎯 Что умеет

- Создаёт вирусные твиты и треды на основе проверенных формул
- Исследует успешные примеры в вашей нише перед написанием
- Использует фреймворки копирайтинга (PAS, AIDA, BAB)
- Оптимизирует под алгоритм X 2026 года
- Генерирует хуки, которые останавливают скролл
- Структурирует треды для максимального вовлечения

## ⚙️ Как работает

### Процесс создания твита:

**Фаза 1: Исследование ниши (КРИТИЧНО)**

Перед написанием ЛЮБОГО твита скилл исследует вирусные примеры в вашей нише:

1. **Определяет нишу/тему** - о чём вы пишете?
2. **Ищет вирусные примеры** - используя web_search:
   - `"[ниша] viral tweet examples"`
   - `"[тема] twitter thread went viral"`
   - `"[топик] best performing tweets"`
3. **Анализирует паттерны**:
   - Какие хуки сработали
   - Структура контента
   - Тон и стиль
   - Конкретные цифры/результаты
   - CTA которые сработали
4. **Документирует инсайты** - создаёт краткий анализ перед написанием

**Фаза 2: Создание контента**

Использует фреймворки и формулы для создания твита/треда.

## 🎯 Алгоритм X (2026)

Понимание алгоритма = ключ к охватам.

### Иерархия вовлечения (от самого важного к менее важному):
1. **Replies (ответы)** - самый весомый сигнал
2. **Quote tweets (цитаты)** - высокая ценность, показывает что ваш контент провоцирует дискуссию
3. **Bookmarks (закладки)** - сильный сигнал ценности
4. **Retweets (ретвиты)** - сигнал усиления
5. **Likes (лайки)** - базовое вовлечение

### Временная чувствительность:
- **Первый час критичен** - если не набрали трекшн за 60 минут, охваты падают
- **Пиковые часы**: 9-11 утра и 7-9 вечера по МСК в будни, 9-11 утра в выходные
- Свежий контент в приоритете - X награждает новизну

### Время просмотра (Dwell Time):
X отслеживает сколько пользователи проводят на вашем контенте. Дольше = больше охватов.
- Треды естественно увеличивают время просмотра
- Визуальный контент держит внимание дольше
- Цепляющие хуки останавливают скролл

### Форматы с бустом:
- **Нативное видео**: приоритет над внешними ссылками
- **Картинки/карусели**: 2x вовлечения vs только текст
- **Треды**: 3x вовлечения vs одиночные твиты
- **Опросы**: высокая вовлечённость

### Что ИЗБЕГАТЬ:
- **Внешние ссылки**: сильно снижают охваты (особенно для не-Premium)
- **Общий контент**: нет отличия = нет охватов
- **Просьбы о вовлечении**: "Лайк и RT" режут охваты

## 🎣 Формулы хуков (самый важный элемент)

Хук определяет 80-90% успеха твита. У вас ~1 секунда чтобы остановить скролл.

### Смелое заявление:
```
"Никто об этом не говорит, но..."
"Непопулярное мнение: [спорное утверждение]"
"Всё что вам говорили про [X] - неправда."
"[Общее убеждение] - миф. Вот правда:"
```

### Конкретный результат:
```
"Я [результат] за [срок]. Вот как:"
"[Число] [достижение] за [время]. Разбор:"
"От [плохое состояние] до [хорошее] за [время]. Тред:"
```
**Пример**: "Я вырос с 0 до 50K подписчиков за 90 дней. Вот точный план:"

### Разрыв любопытства:
```
"Я нашёл [прилагательное] хак про [тему] о котором никто не говорит..."
"Единственная вещь в которой ошибаются [тип людей] про [тему]"
"Почему большинство проваливаются в [X] (и как это исправить)"
```

### Хук-вопрос:
```
"Хотите узнать настоящий секрет [X]?"
"Что если всё что вы знали про [X] было неправдой?"
"Задумывались почему [общая проблема]?"
```

### Хук-история:
```
"3 года назад я был [плохое состояние]. Сегодня я [хорошее состояние]."
"Я почти бросил [X]. Потом случилось это:"
"История о том как я [достижение] (с бюджетом $0):"
```

### Разрыв шаблона:
```
"Все говорят [X]. Они неправы."
"Перестаньте делать [общая практика]. Делайте это вместо:"
"Удалите [общую вещь]. Вот почему:"
```

### Обещание списка:
```
"[Число] [вещей] которые [выгода] (тред):"
"[Число] уроков из [опыт/достижение]:"
"[Число] [категория] которые я хотел бы знать раньше:"
```
**Пример**: "7 AI инструментов которые сэкономили мне 20+ часов на прошлой неделе:"

## 📝 Форматы твитов которые идут виралом

### Формат 1: Списковый (Listicle) - самое высокое вовлечение:
```
Хук: "[Число] [вещей] которые [выгода]:"

1. [Пункт] - [Краткое объяснение]
2. [Пункт] - [Краткое объяснение]
...
[CTA или резюме]
```

### Формат 2: Спорное мнение:
```
Хук: "[Популярное убеждение] неверно."

Вот почему: [2-3 предложения объяснения]

Что работает на самом деле: [Ваша альтернатива]
```

### Формат 3: До/После:
```
[Период времени] назад: [Плохое состояние]
Сегодня: [Хорошее состояние]

Разница? [Один ключевой инсайт]
```

### Формат 4: Фреймворк:
```
Хук: "Фреймворк [Название] для [Результат]:"

Шаг 1: [Действие]
Шаг 2: [Действие]
Шаг 3: [Действие]

[Опционально: краткое расширение каждого]
```

### Формат 5: "Заполни пропуск":
```
"Самый недооцененный навык для _____ это _____."
"Если бы я мог использовать только один инструмент для [X], это был бы _____."
```
*Генерирует массу ответов*

### Формат 6: Универсальный опыт:
```
"Когда ты наконец [общий опыт/осознание]"
"Почему никто не говорит про [общая проблема]?"
"Тот момент когда [узнаваемая ситуация]"
```

## 🧵 Структура треда (идеал - 7 твитов)

### Шаблон треда:

**Твит 1 (Хук)**:
- Самый цепляющий инсайт или результат
- Включите конкретные цифры
- Укажите что это тред: "🧵" или "(тред)"

**Твит 2 (Контекст)**:
- Развейте хук
- Объясните почему это важно
- Создайте больше любопытства

**Твиты 3-6 (Основная ценность)**:
- ОДИН ключевой инсайт на твит
- Используйте нумерацию (1/, 2/, и т.д.)
- Добавляйте визуальные разрывы каждые 3-4 твита (картинки, графики)
- Каждый твит должен быть ценным сам по себе

**Твит 7 (Мост/Резюме)**:
- Суммируйте ключевые выводы
- Соедините с более широким применением

**Твит 8 (CTA)**:
- Задайте вопрос (генерирует ответы)
- Процитируйте первый твит (драйвит ретвиты)
- Направьте на профиль/рассылку

### Правила написания тредов:
1. Каждый твит должен заработать следующий клик
2. Никакой воды - каждое слово должно нести вес
3. Короткие предложения (до 250 символов на твит)
4. "Ваши слова должны читаться как скользкая горка"
5. Нумеруйте твиты (2/12, 3/12, и т.д.)

## 💡 Фреймворки копирайтинга для твитов

### PAS (Проблема - Усиление - Решение)
**Самая надёжная формула для вовлечения**

```
[Проблема]: Вы в [конкретная ситуация]
[Усиление]: И это стоит вам [последствия]
[Решение]: Вот что работает: [ваш ответ]
```

### AIDA (Внимание - Интерес - Желание - Действие)
**Лучше для промо-контента**

```
[Внимание]: Хук который останавливает скролл
[Интерес]: "Вот что большинство не понимает..."
[Желание]: "Представьте если бы вы могли [выгода]"
[Действие]: "Напишите мне [X] чтобы начать"
```

### BAB (До - После - Мост)
**Лучше для историй трансформации**

```
[До]: Я был в [плохое состояние]
[После]: Теперь я в [хорошее состояние]
[Мост]: Разница? [Ваш инсайт/решение]
```

## 🎯 Принципы убеждения

Применяйте их чтобы сделать любой твит более убедительным:

**Конкретность** - "23% рост" лучше чем "большой рост"
- Цифры добавляют достоверности
- Конкретные сроки добавляют срочности
- Детали делают утверждения правдоподобными

**Социальное доказательство** - "500+ клиентов" лучше чем "много клиентов"
- Результаты реальных людей
- Количество пользователей/подписчиков
- Узнаваемые имена/бренды

**Разрыв любопытства** - Создайте информационную асимметрию
- Намекните на ценную инфо не раскрывая всё
- Обещайте конкретные результаты
- Используйте "Вот что большинство пропускает..."

**Спорность** - Бросайте вызов существующим убеждениям
- "Популярное мнение неверно"
- Спорные взгляды получают вовлечение
- Избегайте оскорблений - цельтесь в провокацию мысли

**Узнаваемость** - Общий опыт резонирует
- "Когда ты понимаешь..."
- Универсальные проблемы
- Общие точки пути

## 🚀 Growth хаки

### Стратегия 30 дней на подтему:
Выберите ОДНУ узкую подтему в вашей нише. Постите ТОЛЬКО про неё 30 дней подряд.

**Пример**: Если вы в маркетинге, фокусируйтесь только на "темах писем" целый месяц.

Результат: алгоритм X классифицирует вас как авторитета в этой подтеме.

### Стратегия ответов:
Фокусируйтесь на генерации ответов вместо лайков/ретвитов.
- Задавайте вопросы
- Создавайте твиты с пропусками
- Постите "горячие мнения" которые приглашают к дискуссии
- Алгоритм видит вас как провокатора диалога

### Окно вовлечения:
- Постите 3-5 раз в день
- Вовлекайтесь с 20+ аккаунтами ежедневно (осмысленные ответы)
- Отвечайте на комментарии под вашими постами в первый час

### Правило 80/20:
- 80% чистая ценность (без промо)
- 20% промо-контент
- Ценность в первую очередь строит доверие которое конвертит

## 📏 Рекомендации по длине твитов

- **Одиночные твиты**: до 110 символов работают лучше всего
- **Твиты треда**: до 250 символов каждый
- **Почему короткие работают**: легко сканировать, есть место для quote tweets, оптимизированы под мобильные

---

## ⚠️ Типичные ошибки которых нужно избегать

**Слишком общо** - "Советы для успеха" → "3 шаблона холодных писем которые принесли мне 10 встреч на этой неделе"

**Нет хука** - Начинать с контекста вместо удара

**Просьбы о вовлечении** - "Лайк и RT!" режет охваты

**Внешние ссылки в основном твите** - Помещайте ссылки в ответы

**Нет конкретных цифр** - "Я быстро вырос" vs "Я вырос на 12,847 подписчиков за 63 дня"

**Слишком продажно** - Низкое соотношение ценности, ощущается как промо

**Нет CTA** - Тред заканчивается без чёткого следующего шага

## ✅ Чеклист перед публикацией

Перед постом проверьте:

- [ ] Хук останавливает скролл (смелый/конкретный/интригующий)
- [ ] Первые 7 слов зарабатывают остаток твита
- [ ] Включены конкретные цифры где уместно
- [ ] Внутри лимита символов (110 для одиночного, 250 для треда)
- [ ] Нет внешних ссылок в основном твите
- [ ] Чёткий CTA или драйвер вовлечения
- [ ] Постится в пиковые часы
- [ ] Готовы вовлекаться в ответы в первый час
