# Weather - Прогноз погоды 🌤️

## 📦 Установка

1. Скопируйте папку `skills/weather/` в ваш workspace OpenClaw
2. Убедитесь что `curl` установлен (обычно есть по умолчанию)
3. OpenClaw автоматически подхватит скилл при следующем сообщении
4. Проверка: спросите агента "Какая сейчас погода в Москве?"

## 🎯 Что умеет

- Текущая погода в любой точке мира
- Прогноз на несколько дней
- Детали: температура, влажность, ветер, осадки
- Иконки погодных условий
- Работает БЕЗ регистрации и API-ключей

## ⚙️ Как работает

Скилл использует два бесплатных сервиса:

### wttr.in (основной)

**Краткий формат (одна строка):**
```bash
curl -s "wttr.in/Москва?format=3"
# Вывод: Москва: ⛅️ +8°C
```

**Компактный формат (с деталями):**
```bash
curl -s "wttr.in/Москва?format=%l:+%c+%t+%h+%w"
# Вывод: Москва: ⛅️ +8°C 71% ↙5км/ч
```

**Полный прогноз:**
```bash
curl -s "wttr.in/Москва?T"
# Вывод: Таблица с прогнозом на несколько дней
```

**Коды форматирования:**
- `%c` - погодные условия (иконка)
- `%t` - температура
- `%h` - влажность
- `%w` - ветер
- `%l` - название места
- `%m` - фаза луны

**Опции:**
- `?m` - метрические единицы (°C, км/ч)
- `?u` - американские единицы (°F, mph)
- `?1` - только сегодня
- `?0` - только текущий момент
- `.png` - вывод в PNG картинку

**Примеры:**
```bash
# Нью-Йорк (пробелы заменить на +)
curl -s "wttr.in/New+York?format=3"

# Код аэропорта
curl -s "wttr.in/JFK?format=3"

# PNG картинка
curl -s "wttr.in/Берлин.png" -o /tmp/weather.png
```

### Open-Meteo (резервный, JSON)

Бесплатный API для программного доступа:

```bash
curl -s "https://api.open-meteo.com/v1/forecast?latitude=51.5&longitude=-0.12&current_weather=true"
```

Возвращает JSON с:
- temperature (температура)
- windspeed (скорость ветра)
- weathercode (код погодных условий)

Нужно знать координаты города (можно найти в интернете).

Документация: https://open-meteo.com/en/docs

## 🔧 Настройка под себя

### Выбор формата вывода:

- **Краткий** (`?format=3`) - для быстрых ответов в чате
- **Компактный** (`?format=%l:+%c+%t+%h+%w`) - с деталями в одну строку
- **Полный** (`?T`) - таблица с прогнозом
- **Картинка** (`.png`) - визуальный прогноз

### Единицы измерения:

- Добавьте `?m` для метрических (°C, км/ч) - по умолчанию
- Добавьте `?u` для американских (°F, mph)

### Язык:

wttr.in автоматически определяет язык по IP или можно указать `?lang=ru`

### Города с пробелами:

Замените пробелы на `+` или `%20`:
- ✅ `wttr.in/New+York`
- ✅ `wttr.in/New%20York`
- ❌ `wttr.in/New York`

### Домашний город по умолчанию:

Можно настроить wttr.in на ваш город по умолчанию через cookie или просто запоминать город в конфигурации скилла.
