# Стандарт оформления PDF-инструкций для {{PAID_GROUP_NAME}}

> Эталон: AI-OPS-002 (Подключение Claude Max, март 2026)

---

## Общие параметры

- Формат: A4, портретная ориентация
- Фон: белый (#FFFFFF)
- Поля: 25-30 мм по бокам, 20 мм сверху, 25 мм снизу
- Нумерация: серия AI-OPS-NNN

---

## Титульная страница

### Тёмный баннер (верх, 35-40% высоты)
- Фон: тёмный серо-синий (#3D4553 / #404B5A), НЕ чёрный
- На всю ширину (без полей)
- Текст по центру:
  - **"{{PAID_GROUP_NAME}}"** - bold, белый, ~36-42pt, sans-serif
  - **"Документ AI-OPS-NNN"** - regular, белый, ~12-14pt

### Белая часть (низ)
- Синяя линия-разделитель по центру (~30-40% ширины, цвет #4A90D9)
- Название документа - bold, ~24-28pt, тёмный, по центру
- "Версия X.0 | Месяц YYYY" - серый, ~11-12pt
- Описание - 1-2 строки, ~12pt
- Дисклеймер (если нужен) - приглушённый красный (#C0392B), ~10-11pt

---

## Хедер (стр. 2+)

- Текст: "{{PAID_GROUP_NAME}} | AI-OPS-NNN | Название"
- Серый (#999999), ~8-9pt, по центру
- Под хедером тонкая серая линия (0.5pt)

## Футер

- Номер страницы: "- N -", серый, ~9-10pt, по центру

---

## Заголовки

### H1 (заголовки разделов)
- Формат: "1. Название раздела"
- Bold, ~24-28pt, тёмный (#1A1A2E)
- ПОД заголовком - синяя линия (#4A90D9), ~20-25% ширины, 2-3px, выровнена ВЛЕВО

### H2 (подзаголовки)
- Bold, ~16-18pt, тёмный
- Без декоративной линии

---

## Основной текст

- Sans-serif (Inter / Noto Sans / Source Sans Pro)
- Regular, ~11-12pt, тёмно-серый (#333333)
- Межстрочный: 1.4-1.6
- Выравнивание: по ширине (justified)
- Абзацы отделяются пустой строкой (без красной строки)

---

## Списки

- Маркер: точка (•), чёрная
- Отступ текста: ~10-15 мм от левого поля
- Интервал между пунктами: ~4-6 мм

---

## Блоки кода

- Фон: #F5F5F5
- Рамка: 1px, #E0E0E0, скругление 4-6px
- Padding: 12-16px
- Шрифт: monospace (Source Code Pro / Fira Code), ~10-11pt
- Без синтаксической подсветки

---

## Callout-блоки (информационные)

Все: светлый фон + цветная полоса слева (3-4px) + тонкая рамка остальных сторон.

| Тип | Левая полоса | Фон | Когда |
|-----|-------------|-----|-------|
| ⚠️ Warning | Оранжевый #E8A838 | #F8F8F8 | Предупреждения |
| ✅ Success | Зелёный #4CAF50 | #F0FAF0 | Готово, рекомендация |
| ℹ️ Info | Синий #4A90D9 | #F0F4FA | Справочная информация |

---

## Чего НЕ должно быть

- Тёмная тема
- Фоновые изображения, градиенты, текстуры
- Иконки и иллюстрации
- Жёлтый текст
- Длинное тире - только дефис (-)
- Канцелярит
