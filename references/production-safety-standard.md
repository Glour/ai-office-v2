# Production Safety Standard

> Обязательный стандарт для ВСЕХ проектов команды.
> Уолтер читает ПЕРЕД каждым проектом. Сол проверяет на ОТК ПО этому стандарту.

---

## 1. Бэкап - ПЕРВЫЙ шаг

Любой AGENT.md начинается с предложения бэкапа.

- Спросить разрешение у пользователя
- БЕЗ разрешения НЕ делать бэкап
- Команда: `cp -r ~/.openclaw/agents/main/agent/ ~/agent-backup-$(date +%Y%m%d)/`
- Если отказался - предупредить и продолжить

## 2. НЕ перезаписывать критичные файлы

Файлы MEMORY.md, AGENTS.md, TOOLS.md - только ДОПИСЫВАТЬ (append).
НИКОГДА не перезаписывать с нуля. Потеря памяти и настроек = катастрофа.

## 3. Личные данные - ноль в MD

- В MD файлах: 0 персональных данных (имена, ID, адреса, IP, города)
- В PDF на странице "Ресурсы": ссылки YouTube/Telegram/GitHub - ОСТАВИТЬ
- Telegram ID: placeholder `[TELEGRAM_ID]`
- Пути: `<ваш-workspace>` или generic

## 4. Model-agnostic

- Никаких привязок к конкретной модели (Claude, GPT, Gemini)
- Инструкции работают с ЛЮБЫМ LLM-агентом
- Нет `anthropic/claude-...` в пользовательских файлах

## 5. Кросс-платформа

- macOS + Linux + Windows/WSL
- Определение ОС автоматически через `uname -s`
- Команды с ветвлением по платформе где нужно
- stat/lsof/ss/systemctl - с альтернативами

## 6. Обращение на "вы"

К пользователю/подписчику ВСЕГДА на "вы". Без исключений.

## 7. Дефис (-) вместо тире (-)

Везде. В MD. В PDF. В постах. Точка.

- ~~"данный"~~ (вместо "этот")
- Полный список - в voice-dictionary копирайтера

## 10. Идемпотентность

Любой AGENT.md можно запустить повторно без поломок:
- Перед каждым шагом: проверь, не сделано ли уже
- Если сделано - пропусти, напиши "уже настроено"
- Нет побочных эффектов от повторного запуска

## 11. Визуальная проверка PDF (ОБЯЗАТЕЛЬНО!)

Перед сдачей PDF - ОТКРЫТЬ и ПРОЧИТАТЬ каждую страницу:
- Таблицы: текст не наезжает на соседние ячейки, не обрезается
- Код-блоки: строки не вылезают за границы блока
- Длинные слова: не сливаются, не накладываются друг на друга
- Кириллица: все символы отображаются корректно
- Заголовки: не отрываются от содержания (не висят внизу страницы одни)
- Нумерация: последовательная, без пропусков

Метод проверки: `open файл.pdf` (macOS) или скриншот каждой страницы.
Если нет возможности открыть PDF - извлечь текст и проверить ширину строк в таблицах.

**Нельзя сдавать PDF который ты не открыл и не прочитал глазами.**

## 12. Откат

Инструкция по удалению/откату - ОБЯЗАТЕЛЬНА в каждом продукте.
Человек должен мочь вернуть всё как было.
