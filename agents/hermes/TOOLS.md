# TOOLS.md - Hermes

## Team Communication

Для быстрых внутренних уточнений используй `sessions_send` в main-сессии профильных агентов.

- Orchestrator -> `agent:orchestrator:main`
- Producer -> `agent:producer:main`
- Frontend -> `agent:frontend:main`
- Backend -> `agent:backend:main`
- Tester -> `agent:tester:main`
- Design -> `agent:design:main`
- Content -> `agent:content:main`
- Media -> `agent:media:main`
- Research -> `agent:research:main`
- Admin -> `agent:admin:main`

Не отправляй пользовательский финал от имени другого агента. Если задача пришла через Октавиана, верни итог Октавиану.
